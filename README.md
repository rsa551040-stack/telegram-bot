# Telegram Moderation Bot

بوت تيليجرام جاهز لإدارة المجموعات وحذف الرسائل الإعلانية والكلمات المسيئة، مع **حظر تلقائي** لمن يرسل الإعلانات أو الروابط أو أرقام الهواتف.

## المميزات

- حذف الرسائل الإعلانية تلقائيًا
- حظر المستخدم الذي يرسل إعلانًا
- حظر الرسائل التي تحتوي على:
  - روابط
  - أرقام هواتف
  - العبارات الإعلانية التي طلبتها
- حذف الكلمات المسيئة
- إمكانية جعل الكلمات المسيئة تؤدي إلى الحظر بعد عدد معين من المخالفات
- يعمل داخل المجموعات والسوبر جروب

## العبارات المحظورة المضافة

تمت إضافة العبارات التالية ضمن الحظر:

- ♨️حل واجبات او بحوث او عروض ومشاريع وحتى حلول الكويزات والاختبارات
- حل واجبات او بحوث او عروض ومشاريع وحتى حلول الكويزات والاختبارات
- حل واجبات
- حل بحوث
- حلولهم ثقة ومضمونة ولديهم تجارب العديد من الطلاب والطالبات
- للتواصل واتساب
- تعال خاص
- للتواصل
- خصم
- خدمات طلابية
- سكليف
- مساعدة
- خش خاص
- خاص

> **مهم:** كلمة `خاص` واسعة جدًا وقد تؤدي إلى حذف رسائل عادية أيضًا. إذا أردت، يمكنك حذفها أو تضييقها لاحقًا من ملف `settings.py`.

## المتطلبات

- Python 3.10+
- توكن بوت تيليجرام من BotFather

## الإعداد السريع

### 1) تثبيت المكتبات

```bash
pip install -r requirements.txt
```

### 2) تجهيز ملف البيئة

انسخ الملف:

```bash
cp .env.example .env
```

ثم ضع التوكن داخل `.env`:

```env
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

### 3) تشغيل البوت

```bash
python main.py
```

## مهم جدًا داخل تيليجرام

حتى يعمل البوت على حذف الرسائل وحظر الأعضاء، يجب أن:

1. تضيفه كمشرف داخل المجموعة.
2. تمنحه صلاحيات:
   - Delete messages
   - Ban users
3. توقف **Privacy Mode** من BotFather حتى يتمكن من قراءة رسائل الأعضاء داخل المجموعة.

## الأوامر

- `/start` : التحقق من أن البوت يعمل
- `/ping` : اختبار سريع
- `/status` : عرض حالة الإعدادات

## تخصيص الكلمات المسيئة

افتح ملف `settings.py` وأضف الكلمات المسيئة داخل:

```python
CUSTOM_BANNED_ABUSE_TERMS = [
    "كلمة1",
    "كلمة2",
]
```

## التحكم في الحظر عند الكلمات المسيئة

داخل ملف `.env`:

```env
ABUSE_BAN_THRESHOLD=0
```

- إذا كانت القيمة `0` فلن يتم الحظر بسبب الكلمات المسيئة، وسيتم فقط حذف الرسالة.
- إذا جعلتها مثلًا `3` فسيتم حظر المستخدم بعد 3 مخالفات إساءة.

## النشر على استضافة أو سيرفر
